# Time Based Forward Movement

## Objective
Move TurtleBot3 forward for 5 seconds and stop automatically using ROS2 and Python.

## Tools Used
- ROS2 Humble
- Gazebo
- TurtleBot3 Burger
- Python

## Description
This project publishes velocity commands to the `/cmd_vel` topic. The robot moves forward for 5 seconds and then stops automatically based on elapsed time.

## Result
The robot successfully moved forward for 5 seconds and stopped without manual intervention.
